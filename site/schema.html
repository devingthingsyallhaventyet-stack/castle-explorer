Status Tag Rules

Only these 4 values. If unsure, hide the tag.

  • Open to Visitors ??? large well-maintained properties with admission cost, gift shops, guided tours, set hours
  • Book to Experience ??? hotels, event venues, tours by appointment only
  • Freely Accessible ??? places that can be accessed for free at any time (open ruins, free buildings, listed as free)
  • Private ??? privately owned, not open to visitors, listed to preserve history. Only used if explicitly stated on Wikipedia or a dedicated website.

Condition Tag Rules

Only these 2 values. If unsure, hide the tag.

  • Intact ??? has a roof, well preserved, people living in it or actively maintaining it
  • Ruin ??? no roof, partially standing, regardless of whether free or managed

Badge Rules

Mutually exclusive, highest tier match wins. Data from Google Places.

  • ??? Must See ??? 4.5+ stars AND 5,000+ reviews
  • ???? Hidden Gem ??? 4.0+ stars AND 20???100 reviews
  • ???? Mysterious ??? any rating, fewer than 10 reviews
  • No badge ??? everything else

Gallery Rules

  • 5 images per gallery
  • Google Places photos are the primary source ??? fetched live via API on every page load. Cannot be stored on R2/CDN. Cannot cache photo names (they expire). No curation possible ??? we display whatever Google returns.
  • Wikimedia Commons fills gaps ??? if a listing has fewer than 5 Google photos, remaining slots filled with Wikimedia images hosted on R2
  • Browse/collection pages use Wikimedia thumbnails from R2 only (not Google photos) to control API costs
  • Future: curated galleries ??? as the site grows, we can replace Google photos with hand-picked images (our own, CC-licensed, heritage body press images, etc.) on a case-by-case basis for key listings

Google Places Photo Attribution (Required)

  1. Author name displayed from authorAttributions.displayName, linked to authorAttributions.uri when available
  2. Google Maps logo or "Google Maps" text must appear near the photos
    • Logo: official assets, 16???19dp height, 10dp clearspace (left/right/top), 5dp bottom
    • Text alternative: Roboto font, 12???16sp, normal weight, color white/#1F1F1F/#5E5E5E
  3. Visual distinction ??? Google content must be visually separated from non-Google content (border, background, or whitespace)

Wikimedia Commons Photo Attribution (CC BY-SA)

  1. Author name ??? linked to the original Commons file page (commons.wikimedia.org/wiki/File:...)
  2. License name ??? linked to the license deed (e.g. https://creativecommons.org/licenses/by-sa/3.0/)
  3. Indication of changes ??? if cropped/resized, note it. If unmodified, not required.

Format: ???? [Author Name](commons-file-link) ?? [CC BY-SA 3.0](license-link)

Site-Wide Requirements (Google API Policy)

  • Publicly accessible Terms of Use incorporating Google's ToS
  • Publicly accessible Privacy Policy incorporating Google's Privacy Policy
